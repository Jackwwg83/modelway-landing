隐私政策

最后更新日期：2026-04-20 · 适用于 ModelWay（modelway.dev）及所有子域名

一句话总结：我们只收集让你账户和计费正常工作所必需的数据；我们的服务器仅透传你的 API 请求到 Anthropic / OpenAI / Google Gemini，不留存 prompt 内容或模型回复。

1. 服务提供方

本隐私政策由 ModelWay 团队（以下简称"我们"）发布。ModelWay 是一个 AI API 网关服务，后端基础设施部署在 AWS us-east-1（美国弗吉尼亚北部）。你访问 modelway.dev 及其子域名即表示接受本政策。

2. 我们收集什么

2.1 账户信息

  • 邮箱地址：用于注册、登录、找回密码、发送账单与服务通知
  • 密码哈希：使用 bcrypt 加盐存储，我们无法还原你的明文密码

2.2 使用数据

  • 登录 IP 与 User-Agent：用于风控、异地登录检测、安全审计
  • API 调用元数据：请求时间戳、调用的模型名、input / output token 数、路由到的上游账户标识、响应延迟、HTTP 状态码
  • 计费流水：充值记录、每次调用扣费金额、余额变动

2.3 支付信息

  • 支付订单号、支付渠道（支付宝 / 微信 / Stripe）、金额、币种
  • 我们不接触也不存储你的银行卡号、CVV、支付宝/微信完整账号。支付处理完全由第三方渠道完成，我们仅接收支付成功/失败的回调。

2.4 我们明确不收集的内容

  • 你发给模型的 prompt 内容（请求 body 中的 messages / contents / prompt 字段）
  • 模型返回给你的 response 内容
  • 你的联系人、通讯录、地理位置、设备硬件信息

请求与响应仅在内存中透传到上游和回传给你，不落盘、不进数据库、不进日志。我们的日志系统被严格配置为仅记录元数据（见 2.2）。

3. 数据如何存储

数据类型存储位置存储期限
账户基本信息AWS RDS（us-east-1，加密）账户有效期内
计费与账单流水AWS RDS（us-east-1，加密）5 年（中国税法 / 会计法要求）
API 调用元数据AWS RDS + 归档 S3详细记录 90 天，聚合数据永久
登录日志 / 安全日志AWS CloudWatch90 天
Prompt / Response 内容不存储—

所有静态数据使用 AWS KMS 管理的密钥加密（AES-256），传输全程使用 TLS 1.2+。

4. 第三方共享

为了完成你的 API 调用，我们必须将请求转发给以下上游模型提供商。你的 prompt 内容会到达这些上游，由它们各自的隐私政策处理：

  • Anthropic（Claude 系列）— 隐私政策：anthropic.com/legal/privacy
  • OpenAI（GPT 系列）— 隐私政策：openai.com/policies/privacy-policy
  • Google（Gemini 系列）— 隐私政策：policies.google.com/privacy

我们只转发 API 请求体本身，不会把你的账户信息、邮箱、IP 额外附加给上游。

支付环节，支付订单元数据会传给支付宝 / 微信支付 / Stripe，用于完成交易。

除此以外，我们不会向任何第三方出售、出租、出借你的任何数据。法律要求的司法协助除外（见 §7）。

5. Cookie 与 localStorage

  • mw_session / mw_auth_token：会话凭证，HttpOnly + Secure + SameSite=Lax
  • mw.lang / mw.theme：语言和明暗模式偏好，localStorage，仅本地

我们不使用第三方广告追踪 cookie，不嵌入 Google Analytics、Facebook Pixel 等跨站跟踪。

6. 你的权利

作为 ModelWay 用户，你有以下权利，可通过邮件 support@modelway.dev 行使：

  • 查询权：请求查阅我们持有的关于你的全部数据（3 个工作日内响应）
  • 导出权：以机器可读格式（JSON / CSV）导出账户数据与调用元数据
  • 更正权：要求更正不准确的个人信息（邮箱、账单信息等）
  • 删除权：注销账户并删除全部个人数据（计费流水因税法要求保留 5 年后自动清理）
  • 反对权：反对我们将你的数据用于非必要用途

7. 司法协助与合规

我们遵守中华人民共和国相关法律法规。在收到合法的司法机关出具的调取文件（搜查令、协查函等）时，我们将配合提供所持有的账户元数据。由于我们不存储 prompt 和 response 内容，即使被要求也无法提供。

8. 数据泄露响应

若发生影响用户数据的安全事件，我们将在确认事件性质后 72 小时内通过注册邮箱通知受影响的用户，并说明事件范围、已采取的应对措施和建议行动。

9. 未成年人

本服务面向 18 岁及以上用户。我们不面向未成年人主动营销，若发现未成年人账户我们会主动关停并删除数据。

10. 政策变更

本政策可能随服务演进而变更。变更时我们会：（1）更新本页顶部的"最后更新日期"；（2）下次登录时弹出变更摘要；（3）重大变更（如新增数据类别、变更存储地区）会提前 30 天通过邮件通知。

11. 联系方式

  • 日常咨询：support@modelway.dev
  • 安全漏洞：security@modelway.dev
  • 数据保护相关权利行使：邮件主题请注明「数据请求」
