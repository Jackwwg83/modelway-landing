服务条款

最后更新日期：2026-04-20 · 适用于 ModelWay（modelway.dev）及所有子域名

一句话总结：ModelWay 是一个 AI API 中转网关。你付费调用、我们把请求转发给 Anthropic / OpenAI / Google；你的 prompt 内容、合规性、使用方式由你自己负责，我们不审查也不背书。

1. 本条款生效

你注册 ModelWay 账户或使用其任何服务（包括 API 调用、控制台访问、Claude Code 接入等），即视为你已阅读、理解并同意本服务条款以及同时生效的隐私政策。若你不同意任何条款，请立即停止使用服务并注销账户。

2. 服务说明

ModelWay 提供以下服务：

  • 对 Anthropic、OpenAI、Google Gemini 等上游 AI 模型 API 的统一中转访问（单一 Endpoint、单一 API Key）
  • 按 token 计量的后付 / 预付计费
  • 多账户池调度、亚太边缘优化、故障自动切换
  • 与 Claude Code 等官方客户端的即插即用兼容

服务本质是中转 —— 我们不训练模型，不改写你的 prompt，不替你筛选内容。

3. 你的义务

3.1 遵守上游服务条款

你通过 ModelWay 发起的每一次 API 调用，在法律和上游条款意义上都相当于直接调用上游服务。你必须同时遵守以下各家的 Usage Policies：

  • Anthropic Acceptable Use Policy
  • OpenAI Usage Policies
  • Google Generative AI Prohibited Use Policy

上游因你的调用内容封禁我们的账号池所产生的成本，我们有权向违规用户追偿。

3.2 账户与密钥安全

  • 妥善保管你的登录密码和 API Key，不可分享、不可公开提交到 Git 仓库
  • API Key 泄露导致的调用费用，由你账户承担
  • 发现泄露请立即在控制台吊销并重新生成 Key
  • 一个自然人仅允许注册一个账户

3.3 合法使用

你承诺不将 ModelWay 用于以下行为：

  • 生成、传播违反中华人民共和国法律法规的内容（包括但不限于涉及政治敏感、暴恐、淫秽、赌博、毒品的内容）
  • 侵犯他人知识产权、隐私权、名誉权、肖像权
  • 生成针对特定个人或群体的恶意内容（骚扰、诽谤、勒索、仇恨言论）
  • 生成深度伪造（deepfake）用于欺诈、冒充他人
  • 协助或实施计算机网络攻击、恶意代码开发
  • 大规模自动化爬取、刷量、薅取免费额度
  • 通过技术手段反向工程、绕过计费、绕过配额、干扰我们的调度系统
  • 将服务转售、再分发给第三方（如有商业合作需求请联系 support@modelway.dev）

4. 付费与退款

计费模式：预付充值 · 按 token 调用扣费 · 余额永不过期 · 充值后不支持退款（但你可以继续使用至用完）。

4.1 价格

  • 每次 API 调用按实际 input / output token 数扣费
  • 单价以我们控制台公布为准，会随上游官方调整同步更新
  • 充值促销规则以活动页面为准

4.2 余额

  • 余额计入你的账户，可用于调用任意已开放模型
  • 余额不对应具体 token 数量，汇率随上游价格变动
  • 账户注销时未用完的余额视为放弃

4.3 退款

已充值金额原则上不予退款，以下情形除外：

  • 我们单方面关停服务（见 §7），将按未使用余额原路退还
  • 因我们系统故障导致的计费错误，核实后补偿至余额
  • 30 天内未发起任何 API 调用且可证明充值系误操作的，可申请退至原支付账户（扣除支付渠道手续费）

5. 免责

服务按"现状"提供。我们尽力保证可用性和正确性，但对以下情形不承担责任。
  • 上游故障：Anthropic / OpenAI / Google 等上游停机、限流、改价、下架模型导致你的调用失败或成本变化
  • 不可抗力：地震、战争、政府管制、互联网基础设施中断（海底电缆、骨干运营商等）
  • 跨境网络波动：你到我们边缘节点之间的公网抖动
  • 模型输出质量：模型返回的内容准确性、是否侵权、是否适合你的业务，由上游和使用者共同承担
  • 用户自身失误：API Key 泄露、prompt 写错、死循环导致超额扣费

在法律允许的最大限度内，即使我们因服务故障需要对你承担责任，赔偿总额以你账户内过去 12 个月的累计充值金额为上限。

6. 知识产权

  • ModelWay 的品牌名、logo、网站设计、文档版权属于我们
  • 你通过服务提交的 prompt 与收到的响应，其归属与使用权由上游模型提供商条款决定
  • 你授权我们为提供服务之必要使用你提交的请求元数据（不含 prompt 内容，见隐私政策）

7. 账户终止

7.1 你主动注销

你可在控制台发起账户注销。注销后账户数据按隐私政策处理，未用完余额视为放弃。

7.2 我们关停你的账户

出现以下情况，我们保留立即关停账户、且不退还余额的权利：

  • 违反 §3.3 合法使用条款
  • 触发上游封禁导致我们账号池受损
  • 恶意刷量、反向工程、破坏计费系统
  • 提交虚假身份信息或 60 天内多次违反本条款

7.3 我们整体停止服务

若我们决定关停 ModelWay 整体服务，将提前 30 天通知所有用户，并对未用完余额按原支付渠道退款。

8. 条款变更

本条款可能随业务演进而变更。变更时我们会：

  • 更新本页顶部的"最后更新日期"
  • 在你下次登录时弹出变更摘要供你确认
  • 重大变更（定价机制变动、免责范围变更、管辖地变更等）提前 30 天通过邮件通知

变更生效后继续使用服务，视为接受新版本；若不同意，你有权在生效前注销账户并就未用完余额与我们协商退款。
